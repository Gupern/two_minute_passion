# two_minute_passion
两分钟热度小程序，每次花两分钟，进行锻炼、娱乐和学习，保持热爱，终身进步。

## 微信开发工具配置，进行环境变量配置管理，在详情-本地配置里设置：

- 编译前预处理: `npm run switch:dev`
- 预览前预处理: `npm run switch:dev`
- 上传前预处理: `npm run switch:prod`
